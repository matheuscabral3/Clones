Este projeto tem como intuito calcular o IMC de uma pessoa.

O IMC é reconhecido como padrão internacional para avaliar o grau de sobrepeso e obesidade.
É calculado dividindo o peso (em kg) pela altura ao quadrado (em metros).

O Índice de Massa Corporal é calculado através da equação:

1 - Peso dividido pelo quadrado da altura: Peso / (Altura)² ) = Resultado.
